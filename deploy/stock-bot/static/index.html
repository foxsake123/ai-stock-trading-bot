Get Started in 3 Steps

Register for an API Key

Enter your email below to get your personal API key. This key authenticates your requests.

Configure Claude Desktop

Add this to your Claude Desktop config file:

Windows: %APPDATA%\Claude\claude_desktop_config.json
Mac: ~/Library/Application Support/Claude/claude_desktop_config.json

// Add to your config file: { "mcpServers": { "shorgan-bot": { "url": "https://your-server.com/mcp", "headers": { "Authorization": "Bearer YOUR_API_KEY" } } } }

Connect Your Alpaca Account

Restart Claude Desktop, then connect your brokerage:

# First, connect your Alpaca account "Connect my Alpaca trading account" # Claude will ask for your Alpaca API credentials: # - Go to https://app.alpaca.markets # - Click "API Keys" in the sidebar # - Generate a new key pair # - Paste the API Key and Secret when Claude asks

Note: Start with a paper trading account to test safely with fake money! Your Alpaca credentials are encrypted and stored securely.

Start Trading

Once connected, chat naturally with Claude:

# Check your portfolio "Show me my portfolio" # Get recommendations (safe - no real trades) "Run DEE-BOT in dry run mode" # See what SHORGAN would do "What trades would SHORGAN recommend?" # Execute for real (careful!) "Execute DEE-BOT strategy - I confirm"
